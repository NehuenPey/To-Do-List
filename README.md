# To-Do List – App de Tareas para Móvil
Aplicación móvil de gestión de tareas que permite al usuario organizar su lista de pendientes con funcionalidades completas de creación, edición, eliminación y marcado de tareas como completadas.

Descripción:
- Esta es una app de To-Do List desarrollada para dispositivos móviles donde los usuarios pueden administrar sus tareas de forma intuitiva y eficiente. Cada tarea requiere un nombre mínimo obligatorio, y opcionalmente puede incluir descripción, fecha de realización y categoría para brindar mayor contexto y organización.

La aplicación permite:
- Crear tareas nuevas,
- Editar tareas existentes,
- Eliminar tareas,
- Marcar tareas como completadas.

Funcionalidades:
- Crear nuevas tareas (con nombre obligatorio)
- Agregar descripción opcional a las tareas
- Establecer fecha de realización
- Categorizar tareas por tipo
- Editar tareas
- Eliminar tareas
- Marcar tareas como completadas
- Interfaz simple y enfocada en usabilidad

Tecnologías y herramientas:
- Ionic
- Android Studio
- HTML
- CSS
- JavaScript

Cómo usar:
Desarrollo
1) Instalar:
    - Node.Js
    - Ionic CLI
    - Android Studio
2) Clonar el repositorio: git clone https://github.com/NehuenPey/To-Do-List.git
3) Instalar dependencias: npm install
4) Ejecutar en navegador: ionic serve
5) Ejecutar en emulador Android: ionic cap run android

Uso en celular:
1) Descargar el proyecto.
2) Instalar WinRAR en el dispositivo para descomprimir el archivo.
3) Ejecutar el archivo .exe incluido en el proyecto.
Nota: esta opción está pensada para pruebas locales del proyecto.

Flujo de la app:
- El usuario abre la app.
- Visualiza la lista de tareas existentes.
- Puede agregar una nueva tarea con:
- Nombre (obligatorio)
- Descripción (opcional)
- Fecha de realización (opcional)
- Categoría (opcional)
- Desde la lista puede editar, completar o eliminar cada tarea.

Posibles mejoras:
